    • Lesson 4. Basic HTML and CSS (September 17, 2024)

    Hyper Text Mark up

  • The standard markup language for creating Web pages.
  • Describres the structure of Web pagess using markup.
  • Elements are the building blocks if HTML pages.
  • Elements are presented with tags.
  • Tags label pieces of content such as "heading", "paragraph", "table" and so on.
  • Browser do not display the HTML tags, but use them to render the content of the page.

    • HTML VERSIONS

  • 1. HTML
  • 2. HTML 2.0
  • 3. HTML 3.0
  • 4. HTML 4.0
  • 5. XHTML
  • 6. HTMLS

    • CSS or Cascading Style Sheet

  • Simple design language that describes the style of an HTML. It can be modified according to its color, size, backgrounds, layout and etc.
  • Control the aout of multiple web pages all at once.
  • External Stylesheets are stored in CSS files.
  • Inserting CSS INLINE, INTERNAL, EXTERNAL.
  • Selectors areused to find or select HTML elements.


    • Lesson 5. The Internet and the World Wide Web (November 7, 2024)

    The Internet

  • A global network of interconnected computers and servers.
  • Uses protocols like TCP/IP to transmit data.
  • Provides services such as email, file sharing, and VoIP.

    • The World Wide Web (WWW)

  • A system of linked web pages and multimedia accessed via the Internet.
  • Uses web browsers to display websites.
  • Web pages are created with HTML and accessed using URLs.

    • Key Differences

  • The Internet is the infrastructure, while the Web is a service on top of it.
  • The Web uses the Internet for browsing websites, emails, and more.

    • How They Work Together

  • The Internet connects devices globally, while the Web provides content like websites.
